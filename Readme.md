# GブレイバーAI

## はじめに
自作ゲームのGブレイバー第二弾を作成しています。
まだ作り中なので、環境セットアップ、各種コマンドのメモだけを書いています。

## ローカル開発環境のセットアップ

```
npm -g webpack
npm install
webpack
npm start
```

上記コマンド実行後に、ブラウザから`http://localhost:3000`にアクセスする。

### 各種コマンド

|コマンド                      | 説明                                   |
| npm start                   | 本ゲームを起動する                       |
| webpack                     | フロントエンドのソースコードをビルドする    |
